Description
A TJSValue instance is a reference to a JavaScript value. You use the TJSValue to convert basic values (such as numbers and strings) between JavaScript and Delphi representations in order to pass data between native code and JavaScript code. You can also use this class to create JavaScript objects that wrap native objects of custom classes or JavaScript functions whose implementations are provided by native methods or functions.
